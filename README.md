## Guardian AI. Система проактивного мониторинга редких видов животных

### Описание проекта и проблематика

* Guardian AI — интеллектуальная система, предназначенная для проактивного выявления угроз популяциям редких животных. В заповедниках исчезновение особей часто остаётся незамеченным неделями, что может означать браконьерство, болезни или изменение среды. Система отслеживает активность каждого вида и сигнализирует о рисках ещё до наступления необратимых последствий.


### Принцип работы и технология.

* Распределённая сеть камер с ИИ‑модулями ежедневно распознаёт редких животных и фиксирует их появление с указанием времени и координат. Если конкретный вид ( **Красный волк**, **снежный барс**, **Дальневосточный леопард(Амурский)**, **Амурский тигр**) не попадает в кадр более 14 дней, создаётся сигнал «Пропажа активности». Система передаёт координаты последнего наблюдения инспекторам и запускает протокол проверки с возможностью выезда группы или использования дронов.
Guardian AI превращает охрану природы из пассивного мониторинга в активное реагирование, обеспечивая своевременное обнаружение угроз для редких видов.
